Voici quelques lignes de codes devant permettre d'écrire de manière assez simples des petites applications graphiques en Java. Le code s'appuie sur la bibliothèque standard Java/Swing.

Télécharger

Pour pouvoir utiliser cette bibliothèque, il faut en télécharger le code : DrawingWindow.java. Ce fichier doit être placé au même endroit que vos programmes.

Voici quelques exemples illustrant l'utilisation de la bibliothèque : Hello.java Exemple1.java Exemple2.java Exemple3.java
